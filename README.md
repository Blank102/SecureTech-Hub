# SecureTech Hub 🔐

**SecureTech Hub** is a web platform designed to promote cybersecurity awareness alongside the latest tech news updates. Built with FastAPI, HTML, CSS, and JavaScript, the platform aims to inform users while encouraging safe digital practices.

🌐 **Live Website**: [https://securetech-hub.onrender.com/](https://securetech-hub.onrender.com/)

⚠️ _Note_: This project is hosted on Render's free tier. The site may take **20–30 seconds to load** initially as the server wakes up. Thanks for your patience!

🚧 **Work in Progress**  
SecureTech Hub is currently under development. The **cybersecurity awareness section is yet to be implemented**. New features and content will be added in upcoming updates.

## 🔧 Tech Stack

- **Backend**: FastAPI  
- **Frontend**: HTML, CSS, JavaScript
